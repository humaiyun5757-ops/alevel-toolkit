# A-Level Maths & Physics Toolkit

A Python-based toolkit for solving and exploring key concepts from AQA Physics and Edexcel Maths.

## Features
- SUVAT solver
- Kinetic energy calculator
- Quadratic equation solver
- Projectile motion range calculator

## Motivation
I built this to better understand how mathematical models are applied in physics problems, and to translate exam-based formulas into computational tools.
